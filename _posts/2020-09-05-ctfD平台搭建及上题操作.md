ctfd平台链接 https://github.com/CTFd/CTFd

docker中直接按一下命令操作就可以了.

```
docker search ctfd
docker pull ctfd/ctfd
docker run -id --name=ctfd -p 8000:8000 ctfd/ctfd /bin/bash
```

ok啦。服务器记得开放对应端口。

直接访问 ip:8000

然后设置相应的信息。
