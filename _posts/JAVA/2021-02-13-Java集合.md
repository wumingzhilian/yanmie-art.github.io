## 一、集合

概念：对象的容器，实现了对对象常用的操作，类似数组功能。

和数组区别：

* 数组长度固定，集合长度不固定
* 数组可以存储基本类型和引用类型，集合只能存储引用类型

位置： java.util.*

## 二、Collection 体系集合

![image-20210213121027807](https://gitee.com/luo_fan_1/yanmie-art/raw/master/img/image-20210213121027807.png)



特点：代表一组任意类型的对象，无序、无下标、不能重复。

```java
package com.collection.demo01;

import java.util.ArrayList;
import java.util.Collection;
import java.util.Iterator;

public class Demo01 {
    public static void main(String[] args) {
        // 创建集合
        Collection collection = new ArrayList();
        // 添加元素
        collection.add(1);
        collection.add(2);
        collection.add(3);
        System.out.println(collection.size());  // 3
        System.out.println(collection);         // [1, 2, 3]
        // 删除元素
        collection.remove(2);
        System.out.println(collection);         // [1, 3]
        // 遍历元素
        // 增强 for
        for (Object object : collection) {
            System.out.println(object);
        }
        // 迭代器（专门用于遍历集合的方法）
        Iterator iterator = collection.iterator();
        while(iterator.hasNext()){
            Object o =  iterator.next();
            System.out.println(o);
            //iterator.remove();
        }
        System.out.println(collection.size());
        // 判断
        System.out.println(collection.contains(2));  // false
        collection.clear();
        System.out.println(collection.isEmpty());   // true


    }

}
```

Demo.java

```java
package com.collection.demo01;

import java.util.ArrayList;
import java.util.Collection;
import java.util.Iterator;

public class Demo {
    public static void main(String[] args) {
        Student s1 = new Student("张三",13);
        Student s2 = new Student("李四",20);
        Student s3 = new Student("王五",8);
        Collection collection = new ArrayList();
        collection.add(s1);
        collection.add(s2);
        collection.add(s3);
        System.out.println(collection.size());  // 3
        System.out.println(collection.toString());
        // [com.collection.demo01.Student@1b6d3586, com.collection.demo01.Student@4554617c, com.collection.demo01.Student@74a14482]
        // 删除
        collection.remove(s1);
        System.out.println(collection.size());  // 2
        // 遍历
        Iterator iterator = collection.iterator();
        while(iterator.hasNext()){
            Student s = (Student) iterator.next();
            System.out.println(s.toString());
        }
        // 判断
        collection.clear();
        System.out.println(collection.isEmpty());
    }
}
```

## 三、List 集合

特点：有序、有下表、元素可以重复。

* void add(int index, Object o)      在 index 处插入对象 o
* boolean addAll(int index, Collection c)    将一个集合中的元素添加到此集合中的 index 位置
* Object get(int index)     返回集合中指定位置的元素

* List subList(int fromIndex, int toIndex)   返回参数位置之间的元素

Demo01.java

```java
package com.collection.list;

import java.util.ArrayList;
import java.util.Iterator;
import java.util.List;
import java.util.ListIterator;

public class Demo01 {
    public static void main(String[] args) {
        List list  = new ArrayList();
        list.add("苹果");
        list.add("小米");
        list.add(0,"华为");
        System.out.println(list);   // [华为, 苹果, 小米]
        //list.remove("小米");
        list.remove(2);
        System.out.println(list);  // [华为, 苹果]
        for (int i=0;i<list.size();i++) {
            System.out.println(list.get(i));    // for 遍历
        }
        for (Object object:list) {
            System.out.println(object);     // foreach 遍历
        }
        Iterator iterator = list.iterator();
        while(iterator.hasNext()){
            System.out.println(iterator.next());  // 迭代器 遍历
        }
        System.out.println("-------------------------");
        ListIterator listIterator = list.listIterator();
        while(listIterator.hasNext()){
            System.out.print(listIterator.nextIndex()+":"+listIterator.next()+"\t");
        }
        // 先遍历到后边，才能能后边再往前，指针
        while(listIterator.hasPrevious()){
            System.out.print(listIterator.previousIndex()+":"+listIterator.previous()+"\t");
        }
        System.out.println(list.contains("华为"));   // true
        System.out.println(list.indexOf("华为"));    // 0
    }
}
```
